# Student Registration System (JSP & Servlet)

## Project Description
This project is a simple **Student Registration System** developed using **JSP and Servlets**.  
It allows users to register student details through a web form and displays the registered details on a success page.

---

## Project Objective
- To understand the basics of JSP and Servlet technology
- To learn how to handle form data in a Java web application
- To understand request and response flow in web applications

---

## Technologies Used
- Java  
- JSP (Java Server Pages)  
- Servlets  
- HTML  
- Apache Tomcat Server  

---

## Project Pages
1. **Home Page**
   - Displays the Student Registration Panel
   - Provides a link to the registration form

2. **Register Student Page**
   - Collects student details:
     - Student Id
     - Student Name
     - Student Address
     - Student Average
   - Sends data to the servlet using POST method

3. **Registration Successful Page**
   - Displays registered student details using JSP Expression Language (EL)

---

## Application Flow
1. User opens the Home Page  
2. Clicks on the Register link  
3. Enters student details in the form  
4. Data is sent to the Servlet  
5. Servlet processes the request  
6. Success page displays student details  

---

## Learning Points
- Learned how to create JSP pages
- Learned how to design HTML forms
- Understood form submission using POST method
- Learned communication between JSP and Servlet
- Used Expression Language (`${}`) for dynamic data
- Understood basic MVC architecture
- Learned page navigation in web applications

---

## Conclusion
This project helped in understanding the fundamentals of JSP and Servlets and how they work together to build a simple web-based application.
